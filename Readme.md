# Titanic EDA & ML Foundations
This repository contains the Day 25 assignment focused on Exploratory Data Analysis (EDA) and Machine Learning classifications using Python in Google Colab.

---

### AM Session: EDA & Pandas
* **Dataset:** Titanic Dataset (CSV with >500 rows).
* **Exploration:** Displayed first/last 5 rows, shape, and column names.
* **Cleaning:** Identified and handled missing values in 'Age' and 'Cabin'.
* **Descriptive Statistics:** Interpreted mean, std, min, and max using `.describe()`.
* **Categorical Analysis:** Analyzed unique values and frequency counts for features like 'Sex'.
* **Advanced Tasks:** * Filtered data with multiple conditions (e.g., Age > 30 and Sex).
    * Created new features like `FamilySize`.
    * Sorted data by numerical columns (Fare).
    * Grouped data by `Pclass` to compute average costs.

---

### PM Session: Machine Learning Types
* **Problem Identification:** Classified real-world scenarios:
    * **Email Spam:** Supervised Learning.
    * **Customer Grouping:** Unsupervised Learning.
    * **Robot Walking:** Reinforcement Learning.
* **Regression vs. Classification:** * **Regression:** Predicting continuous values (e.g., House Prices).
    * **Classification:** Predicting discrete labels (e.g., Survived vs. Not Survived).
* **Data Prep:** Separated features ($X$) and target ($y$) variables.
